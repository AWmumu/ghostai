# 截图说明

放在这个目录的图片会被 README 引用。

需要的 5 张：

| 文件名 | 怎么截 |
|---|---|
| `1-landing.png` | https://www.ghostai.top 主页 hero 区（前 800px 高度） |
| `2-pricing.png` | 主页 #pricing 区块（3 套餐卡片） |
| `3-checkout.png` | https://www.ghostai.top/buy/checkout?plan=team&months=3 整页 |
| `4-admin.png` | Dify 控制台首页（登录后看到的工作空间） |
| `5-dify-chat.png` | 创建一个测试知识库 + 问问题的对话截图 |

要求：
- 宽 1200-1600 px 最佳
- PNG 或 JPG（PNG 优先）
- 单文件 < 500 KB（用 `pngquant` / `tinify` 压一下）
- 中文 UI / 不要带敏感数据（如 admin 真实邮箱）

截好后直接放这个目录，文件名严格按上表。git add → push 即可。
