# 快速开始 · GhostAI 知识库

5 分钟从下单到第一个 AI 问答。

## 第 1 步：下单

1. 访问 [www.ghostai.top/buy](https://www.ghostai.top/buy)
2. 选套餐：个人版 ¥39 / 团队版 ¥199 / 企业版 ¥999
3. 选月数（1/3/6/12）
4. 填**真实邮箱**（账号会发到这里，填错收不到）
5. 点"下一步：付款"

## 第 2 步：付款

跳转到付款页：
- 屏幕上显示**订单号**和**应付金额**
- 用支付宝或微信扫码付款
- 付款时**备注里填订单号**（方便核对）
- 付完截图付款页

## 第 3 步：上传付款凭证

回到付款页：
- 点"选择文件"上传付款截图（JPG/PNG，5MB 以内）
- 点"提交付款凭证"
- 页面提示"已提交，admin 审核后开通"

## 第 4 步：等待激活

通常 **1 小时内** admin 核对到账后激活：
- 你的邮箱会收到欢迎邮件
- 内含临时密码 + 登录链接
- 点链接登录 [www.ghostai.top/signin](https://www.ghostai.top/signin)

> 工作时间内（10:00-22:00）一般 10-30 分钟内激活

## 第 5 步：创建第一个知识库

登录后：
1. 顶部菜单 → **知识库** → **创建知识库**
2. 输入名称（如"产品手册问答"）
3. **上传文档**：PDF / Word / Markdown / 网页链接均可
4. 选 indexing 方式（默认"高质量"=向量检索，推荐）
5. 等待 indexing 完成（小文档 30 秒，大文档几分钟）

## 第 6 步：创建对话应用

1. 顶部菜单 → **工作室** → **从模板创建** → "聊天助手"
2. 给应用起名 → 点"创建"
3. 在右侧"上下文"里勾选刚才的知识库
4. 编辑系统提示词（如"你是公司客服，根据知识库回答问题"）
5. 点"发布"

## 第 7 步：用起来

发布后会得到：
- **公开链接**（直接发给客户/员工，他们能问问题）
- **嵌入代码**（粘到企业官网，浮动客服按钮）
- **API 密钥**（团队版以上）

## 常见疑问

### 没收到激活邮件？

1. 检查垃圾邮件箱（163/QQ 邮箱有概率分类错）
2. 看是不是填错了邮箱（在订单系统不能改）
3. 加客服微信反馈（首页 /buy 底部）

### 付款没核对怎么办？

admin 一般 1 小时内处理。超过 2 小时可加客服微信催。

### 想升级套餐？

后台 → **设置** → **订阅** → **升级套餐**（差价补足，不重新计费）

### 数据安全吗？

- 每个客户独立工作区（Dify tenant 隔离）
- 数据存在国内服务器，不出境
- 每日自动备份到本地 NAS，30 天滚动保留
- 文档不会用于训练任何模型
