# 常见问题 · GhostAI 知识库

## 关于产品

### Q1: GhostAI 跟 Dify 官方 Cloud 是什么关系？

GhostAI 内核是 [Dify](https://dify.ai)（开源 LLMOps 平台）+ 国内自托管 + 中文模型 + 我们的定制 SaaS 层（账号管理、计费、客服）。

**关键差异**：
- Dify 官方 Cloud 服务器在海外（数据出境）；GhostAI 在国内
- Dify 官方接 OpenAI / Anthropic；GhostAI 默认 DeepSeek + bge-m3，中文场景优化
- Dify 官方按用量计费有不确定性；GhostAI 套餐制，预算可控

### Q2: 数据安全吗？文档会被用于训练吗？

- **不会**。每个客户独立 workspace（Dify tenant 隔离），数据物理分开
- 数据存在国内服务器，不出境，符合 GDPR / 国密合规
- 我们不读、不导出、不用任何客户数据训练模型
- 每日自动备份到 LAN 内 NAS，30 天滚动保留

### Q3: 用的什么 LLM？能换吗？

**默认配置**：
- 大模型：DeepSeek V4
- 嵌入：bge-m3（BAAI 开源，中文优化）

**可切换**：登录控制台后可配置任意支持 OpenAI 协议的模型：
- OpenAI / Anthropic Claude / Google Gemini（需自备 API key）
- Qwen / GLM / Moonshot / Yi 等国产模型
- 本地部署的 Llama / Qwen（如果你有自己的 GPU 服务器）

### Q4: Token 用完了怎么办？

- 系统会提前在 80% 用量时发邮件提醒
- 用完后停止 AI 对话功能，但**数据保留**
- 续费/升级套餐立即恢复

### Q5: 能私有化部署吗？

可以，**企业版起步**。提供：
- 全套 Docker Compose 部署包
- 1 年技术支持
- 升级 / 补丁通道

[联系商务](mailto:support@ghostai.top) 获取报价。

## 关于使用

### Q10: 支持哪些文档格式？

- 文本：TXT, MD, RTF
- 文档：PDF, DOC, DOCX
- 表格：XLS, XLSX, CSV
- 演示：PPT, PPTX
- 网页：直接粘贴 URL，系统抓取正文
- Notion：导出后上传

### Q11: 单个文档最大多大？

- 100 MB 单文件上限
- 上传后系统自动切块（默认每段 500 token，可调）

### Q12: 知识库能多人共编辑吗？

- **团队版**：3 个账号可同时编辑
- **企业版**：不限账号 + 角色权限（管理员/编辑/只读）

### Q13: 能接入企业微信 / 钉钉 / 飞书吗？

- 团队版以上可用 API 接入
- 我们提供 [示例代码](API.md) 帮你 1 小时内集成

## 故障排除

### Q14: 登录提示账号不存在？

确认：
- 邮箱地址完全正确（含大小写）
- 临时密码没复制错（开头/末尾别带空格）
- 不是其他人的账号（你订单上的邮箱 ≠ 你的常用邮箱时）

### Q15: 文档上传后一直 indexing 中？

可能：
- 文档太大（>50MB），耐心等
- bge-m3 在处理 backlog，多等几分钟
- 文档格式有问题 → 转 PDF 重传

### Q16: 问问题没返回知识库的内容？

检查：
- 应用是否绑定了知识库（右侧"上下文"勾选）
- 知识库 indexing 是否完成（看知识库详情页）
- 问题用词是否跟文档接近（向量检索靠语义，但太宽泛会失败）
